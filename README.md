# Testes End-to-End com Cypress  
### Projeto baseado na aplicação Real World App (RWA)

Este repositório contém exercícios de testes automatizados utilizando Cypress, aplicados na aplicação Real World App, com foco em validar fluxos principais de uma aplicação financeira, como:

- Cadastro de usuários
- Login
- Transferência de dinheiro
- Validação de saldo
- Visualização de histórico de transações

---

## Tecnologias utilizadas

- Cypress
- JavaScript
- Node.js
- Real World App (ambiente local)

## Cenários testados

### ✔ Registro de usuário
- Cadastro com dados válidos
- Validação de campos obrigatórios
### ✔ Login
- Login com credenciais válidas
- Login com credenciais inválidas
### ✔ Transferência de dinheiro
- Envio com saldo suficiente
- Envio com saldo insuficiente
### ✔ Histórico de transações
- Visualização com transações
- Visualização sem transações
