# Casos de Teste — Exercício 3 

**Aplicação:** Real World App  
----

**Nome do Caso:** Visualizar histórico de transações com sucesso  

**Objetivo:** Verificar se o sistema exibe corretamente o histórico de transações do usuário autenticado.  
**Pré-condições:** Usuário autenticado com transações cadastradas.  
**Passos:**  
1. Realizar login.
2. Acessar a aba "Mine". 
**Resultado esperado:** O sistema exibe a lista de transações do usuário com pelo menos um item listado.  

----

**Nome do Caso:** Visualizar histórico de transações com sucesso  

**Objetivo:** Verificar se o sistema exibe corretamente mensagem quando não há transações no histórico do usuário.  
**Pré-condições:** Usuário autenticado sem transações cadastrada.  
**Passos:**  
1. Realizar login.
2. Acessar a aba "Mine". 
**Resultado esperado:** O sistema exibe mensagem informando que não há transações ("No Transactions").
